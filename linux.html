Linux note

週一 04 十一月 2013
By jack

Linux紀錄

Find /etc, ls -al , pwd

file open: nano

編輯軟體: yum install nano ex, nano -w /etc/mysql/my.conf

cat -n /etc/mysql/my.conf ; -n列行號

vi /etc/mysql/my.conf

mysql 權限: chown -R mysql:mysql /var/lib/mysql;

install mysql & run: sudo yum install mysql mysql-server sudo service mysqld start

install PHP: sudo yum install php php-mysql

PHP Modules:

yum search php- Install: sudo yum install of the module

boot auto run by apache & mysql:

sudo chkconfig httpd on sudo chkconfig mysqld on

create info.php:

step1: sudo nano /var/www/html/info.php

step2:(restart the apache) sudo service httpd restart

檔案&目錄權限:

chmod ### ###為數字,ex777 所有權限

說明: owner = rwx; group=rwx; others=rwx; r=:4 , w:2 , x= 1, chmod(r+w+x,r+w+x,r+w+x) :

Disable firewall: systemctl stop iptables.service GUI: system-config-firewall-tui

Install ftp:

yum install vsftpd

setsebool -P ftp_home_dir=1

start ftp service:

systemctl start vsftpd.service

systemctl enable vsftpd.service

FTP 啟動status查詢:

systemctl status vsftpd.service

編輯ftp存取權限:

conf檔案:

/etc/vsftpd/vsftpd.conf (# anonymous_enable=no ;不開放匿名連線)

移除#, 於 chroot_llist_enable=YES ; (使chroot檔作用)

並建立 /etc/vsftpd/chroot_list檔

加入欲限制 user name :wq存檔

reset apache: systemctl restart vsftpd.service

systemctl enable vsftpd.service

Install DHCP:

sudo yum install dhcp

DHCP server configuration file:

/etc/dhcp/dhcp.conf

Restart DHCP:

systemctl restart dhcp.service

Copy sample file to dhcp.conf:

cp /usr/share/doc/dhcp*/shcpd.conf.sample /etc/dhcp/dhcpd.conf

Edit dhcp.conf:

vi /etc/dhcp/dhcpd.conf

systemctl restart dhcpd.service

systemctl enable dhcpd.service

crate folder:

mkdir -m <權限>

remove null dir:

rmdir

remove not null dir:

rm -r

改變 dir/file 擁有者:

chown -R <帳號名or群組名>

#-R為遞迴

改變dir/file所屬群組:

chgrp

Install virtualbox: yum install kernel-PAE-devel dkms virtualbox-4.1
